Mes balises HTML
Nom Balise Description Remarque
html Le tag html permet de définir un document de type HTML x
head Cette balise permet de regrouper diverses informations sur la page. (Utile pour le navigateur) Quelques balises que l'on peut trouver à l'interieur: title, style, link, meta ...
title Permet de définir le titre du document (visible dans l'onglet du navigateur) Cette balise doit être à l'intérieur de la balise head
body Cette balise permet de définir le body du document. C'est dans le body que l'on ajoute le contenu HTML (texte, lien, image, tableau etc)

h1, h2, h3, h4, h5, h6

link Cette balise doit être à l'intérieur de la balise head
